## Python 基礎：串列（list）和元祖（tuple）集合

集合（collection）是用來記錄多重資料值的容器，它使我們能用更有效率的方式組織和操縱資料。在 Python 的集合當中，最基礎的集合之一叫做**串列（list）**：

```
>>> lst = [1, 2, 3, 4, 5]
>>> lst
[1, 2, 3, 4, 5]
```
